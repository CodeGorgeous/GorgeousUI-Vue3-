# Vue3组件文档

### Icon/图标

**Code**
```vue
<template>
    <Cg-Icon value="&#xe6eb;" :ifRotate="true" />
</template>
```

**API**
|属性|说明|类型|可选值|默认值|
|:-:|:-:|:-:|:-:|:-:|
|value|图标代码|String|undefined|undefined|
|size|图标尺寸|Number|undefined|20|
|color|图标颜色|String|undefined|#303030|
|ifRotate|是否自动旋转|Boolean|undefined|true|
